# Тестове завдання React.js (демо сайту: https://bohdan-zhytchenko-test-task.netlify.app/)

## Завдання 1: Створити початковий React app

-створити React app, завантаживши усі необхідні бібліотеки для базової роботи React
-встановити react-router-dom для навігації між сторінками
-видалити зі стартового пакету файлів ті, що не будуть використовуватися
-додати усі константні значення, надані в умові задачі
-створити базовий файл з BrowserRouter та додати туди дві поки пусті сторінки: сторінку користувачів та сторінку редагування користувачів
-створити як окремий компонент кнопку, яка буду слугувати посиланням на сторінку та параметрами прийматиме текст, статус активності сторінки та адресу сторінки
-створити у файлі з роутером хедер з двома створеними кнопками 
-додати базову стилізацію сторінки, яка прослідковується, у css файл: рамку сайту, шрифти, заголовоки та підзаголовки 

## Завдання 2: Створити сторінку Edit Users 

-створити компонент випадне меню (з заголовком) з можливістю обрати лише один варіант відповіді, який прийматиме параметрами масив значень, заголовок, обране значення та функцію для зміни цього значення
-створити елемент поле для редагування значення ім'я
-створити дві кнопки: для скидання змін (весь стан змін буде очищено та повернуто попередні значення) та їх збереження (зміни будуть застосовані до масиву користувачів)
-створити сторінку з завтосуванням створених раніше компонентів

## Завдання 3: Створити сторінку Users

-створити як окремий компонент випадний список, який параметрами прийматиме масив значень, state обраних параметрів та setState для додавання або видалення певних елементів з обраних
-для кожного елемента випадного списку додати можливість змінювати стан обраних елементів
-додати кнопку видалення параметрів пошуку, яка очищатиме стан обраних параметрів
-створити компонент поле користувача, що відображатиме ім'я, відділ, країну, статус користувача та матиме кнопку видалення користувача
-створити таблицю, яка в залежності від стану обраних параметрів пошуку буде відображати поля користувачів
-створити сторінку користувачів з використанням усіх рініше зроблених компонентів, створити кнопку Add User, яка при натисканні змінюватиме стан відображення майбутнього спливаючого вікна

## Завдання 4: Створити спливаюче вікно Add User

-створити пустий компонент спливаючого вікна
-для створення вікна можна скористатися компонентами, створеними для сторінки редагування користувачів: поле та випадним списком, додавши у компоненті випливаючого вікна useState для відстежування зміни компонентів
-додати 2 кнопки: 1 для закриття вікна та скидання useState, 2 для додавання користувача до масиву користувачів та скидання useState

-зробити кастомний домен
